# Epic 5: Signal Enrichment (Social Proof & Growth)

## Overview
Deepen the "Signal" aspect of HighGrowthJobs by ingesting 3rd-party trust and growth metrics.

## Requirements
- Scrapers/API integrations for:
    - G2 Review Scores.
    - Trust Pilot Scores.
- Historical job opening tracking to calculate "Hiring Velocity."
- UI enrichment showing "Growth Momentum" badges.

## Technical Considerations
- Normalization of disparate score scales.
- Periodic background "Janitor" refreshes.
