# Epic 6: Culture Agent (Voice AI)

## Overview
A specialized AI agent that facilitates voice-based realistic interview simulations focused specifically on company culture and mission.

## Requirements
- Real-time WebRTC or WebSocket-based voice interaction.
- Company-specific "Culture Knowledge Base" (Scraped from Glassdoor/Mission statements).
- Feedback loop: Analyzing user responses against company value profiles.

## Technical Considerations
- Integration with LLM (GPT-4o/Gemini) + TTS/STT providers (Deepgram/ElevenLabs).
- Latency optimization for natural conversation.
