# Epic 4: Network Intelligence (LinkedIn Integration)

## Overview
Enable users to leverage their professional network to find "warm" introductions to high-growth companies.

## Requirements
- OAuth2 integration with LinkedIn API.
- Mapping of user connections to discovered startups in the HJG database.
- UI indicator on company cards showing "X people you know work here."

## Technical Considerations
- Rate limiting and API quotas.
- Data privacy and connection caching.
