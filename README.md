# HighGrowthJobs

**AI-native job intelligence platform designed to connect talent with high-growth, VC-backed startups.**

## Overview
HighGrowthJobs is an AI-powered search engine that automates the discovery of roles at companies with high growth potential (VC-backed startups).

## Features
- **Multipass Scraper**: Multi-level scraping strategy (API, Proactive Probing, Static, Browser).
- **Agentic Navigation**: Gemini-powered browsing to find hidden job lists.
- **Janitor Service**: Automatic background synchronization and job deduplication.
